Odoo 13 POS Promotion (Enterprise)

Allows user to get various Promotional Offers on products.

Note: This module is tested and working in Enterprise Edition. And we are supporting Ubuntu OS and not Windows OS.

Available for Community

Features

  • Get Promotional Offers on Products
  • Define Brands on Products.
  • Buy X Item Get Y Item Free.
  • Buy X Item Get Discount on Y Item.
  • Discount(%) on Total Order Amount.
  • Percent Discount on Quantity.
  • Fixed Discount on Quantity.
  • Discount on Conbination of Product.
  • Discount on Multiple Categories.
  • Discount on Above Price.

How to use

  • Click on "Pos Promotion" to set offers.
  • To get Promotional Offers, select any one of the "Promotion Type".

  • Description of all promotional offers are shown in the below images.















  • Applied promotions can be seen as below.

  • Applied promotions can also be seen on the Receipt Screen.

  • To define Brands for products click on "Configuration > Product > Product Brands".
